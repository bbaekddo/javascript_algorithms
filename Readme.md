### 8. 재귀함수와 완전탐색(DFS)
- 재귀함수
- 재귀함수를 이용한 이진수 출력
- 이진트리 순회

> 📌 DFS
>
> - 재귀함수를 이용해서 가장 끝 레벨까지 도달 후 출력
> - 기본적으로 현재값을 선택하거나 선택하지 않은 뒤 다음 레벨로 넘어감
> - 선택한 값을 저장하는 임시 배열 필요

- 부분집합 구하기
- 합이 같은 부분집합
- 바둑이 승차
- 최대점수 구하기
- 중복순열

> 📌 선택지가 2개 이상인 경우
>
> - 이진 트리가 아닌 선택지가 다수일 경우
> - 재귀함수 내 for문이나 while문과 같은 반복문을 추가로 넣고, 선택지 개수만큼 반복
> - 반복문을 넘어가기 전 현재 값 초기화 필요 (아무것도 선택하지 않은 상태로)

- 동전 교환
- 순열 구하기
- 팩토리얼
- 조합의 경우수

> 📌 메모이제이션
>
> - 동적계획법의 방법론 중 하나
> - 이중 배열을 만들어놓고, 미리 구한 값을 저장해놓은 다음 계산할 때 꺼내 쓰는 것
> - 재귀함수로 여러 번 계산할 문제를 빠르게 해결할 수 있음

- 수열 추측하기

> 📌 메모이제이션 활용
>
> - 모든 경우의 수를 구하지 않고, 정답이 도출되는 과정의 규칙을 찾기
> - 메모이제이션을 활용해서 경우의 수를 대폭 줄일 수 있음

- 조합 구하기
- 수들의 조합

### 9. 그래프와 탐색(DFS, BFS)
- 그래프와 인접행렬

> 📌 그래프
>
> - 무방향 그래프일 경우 (start, end) 좌표와 반대인 (end, start) 모두 1 표시
> - 방향 그래프일 경우 (start, end) 좌표만 1 표시
> - 가중치 그래프일 경우 (start, end) 좌표에 가중치값 표시

- 경로탐색 (인접행렬)
- 경로탐색 (인접리스트)

> 📌 인접행렬과 인접리스트
>
> - 인접 행렬은 좌표 배열을 만들고 1과 0을 이용해 그래프를 표시하는 방법
> - 인접 행렬을 사용하면 구현은 간단하나 포인트가 많아질 수록 효율이 매우 낮아짐
> - 인접 리스트는 좌표 배열을 만들되 (n X n) 좌표가 아닌, 실제 방향이 있는 좌표만 표시

- 미로탐색 (DFS)
- 이진트리 BFS

> 📌 BFS
>
> - 최단 거리 문제를 이용할 때 주로 활용
> - DFS는 재귀함수를 이용해서 구현하지만, BFS는 for 반복문 내에서 주로 사용
> - 레벨을 구분해서 문제를 풀 때 사용

- 송아지 찾기
- 섬나라
