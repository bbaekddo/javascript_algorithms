# 📝 자바스크립트 알고리즘 문제풀이

## 강의 개요
### - 인프런
- 강사 : 김태원
- 과정 : 자바스크립트 알고리즘 문제풀이
- 내용 : 코딩테스트 대비용

## 목차
### 1. 기본문제 풀이
- 세 수 중 최솟값
- 삼각형 판별하기
- 연필 개수
- 1부터 N까지의 합
- 최솟값 구하기
> 📌 Math 내장함수 활용
>
> - max, min으로 바로 최댓값, 최솟값 구하기
> - ...array 구문으로 배열 내 요소 나열
- 홀수
- 10부제
> 📌 Array 내장함수 활용
>
> - map : 특정 조건으로 새로운 배열로 반환
> - filter : 특정 조건을 기준으로 배열 분할
> - reduce : 주로 총합 계산할 때 사용
